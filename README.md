# 🏎️ Pit Stop - Sistema de Estacionamento Premium

> Solução inteligente, ágil e sofisticada para a gestão de estacionamentos de alto padrão.

O **Pit Stop** foi inspirado na eficiência e precisão das pistas de corrida. Projetado especificamente para atender às demandas de **shoppings, resorts, hotéis de luxo e eventos VIP**, o sistema garante que a experiência de entrada e saída do cliente seja impecável, segura e extremamente veloz.

## 🌟 Principais Funcionalidades

*   **⚡ Check-in Veloz:** Leitura automatizada de placas (ANPR) e emissão de tickets digitais via QR Code.
*   **🔑 Gestão de Valet (Manobristas):** Atribuição de tarefas em tempo real, controle de chaves e histórico da equipe de manobra.
*   **🏨 Integração Hoteleira:** Vinculação direta da taxa de estacionamento à conta do quarto do hóspede (ideal para resorts e hotéis).
*   **💳 Pagamento Sem Filas:** Integração com PIX, cartões e carteiras digitais direto pelo smartphone do cliente, eliminando os totens de pagamento.
*   **📊 Painel de Controle (Dashboard):** Gráficos em tempo real com ocupação das vagas, faturamento e fluxo de horários de pico.

## 📋 Arquitetura do Sistema

O ecossistema do **Pit Stop** opera em três frentes integradas:

1. **Totem de Entrada/Saída:** Interface rápida e intuitiva para o cliente final.
2. **App do Manobrista:** Agilidade na palma da mão para quem opera os veículos de luxo.
3. **Painel do Administrador:** Visão macro do negócio para gerentes e diretores do estabelecimento.

## 🚀 Como Executar o Projeto

### 📌 Pré-requisitos
Antes de começar, você vai precisar ter instalado em sua máquina o **Git** e o **Node.js**.

### 🤖 Clonando e rodando

```bash
# Clone este repositório
git clone [https://github.com/seu-usuario/pit-stop.git](https://github.com/seu-usuario/pit-stop.git)

# Acesse a pasta do projeto
cd pit-stop

# Instale as dependências
npm install

# Execute a aplicação em modo de desenvolvimento
npm run dev
